# Story Platform

A scalable story writing and consumption platform built with
React, Spring Boot, and MongoDB — designed as a production-grade system
and long-term portfolio project.

## Tech Stack
- Frontend: React 19 (TypeScript, Vite)
- Backend: Spring Boot 4 (Java 25)
- Database: MongoDB Atlas
- Infra: Docker, Nginx, GitHub Actions, Oracle Cloud VM
